
# Mini Hackathon Project

This project was developed during a Mini Hackathon organized by the Saylani Mass IT Training Program. The hackathon's primary goal was to provide hands-on practice while polishing and testing our skills in HTML, CSS, Bootstrap, Netlify, and GitHub.


## Time Duration

2 hours
## Key Features

#### 1) Responsive Design:
 Built with Bootstrap to ensure a responsive layout across various screen sizes.
#### 2) Deployment: 
Hosted using Netlify for easy and quick deployment.

#### 3) Version Control:
 Managed through GitHub for seamless collaboration and version tracking.

## Technologies Used

#### HTML for structure.
#### CSS  for styling.
#### Bootstrap for responsive design and layout.
#### Netlify for deployment.
#### GitHub for version control and project collaboration.
